# PC Assembly Laboratory

## Summary
We gained practical experience assembling a personal computer, covering:  
- CPU, RAM, Motherboard, PSU, Storage devices  
- Operating System installation  

## Challenges
- Installing components correctly without causing damage  
- Proper cable management and connections  

## Personal Reflection
This lab taught us **practical hardware skills**, improved confidence in handling sensitive components, and reinforced the importance of **attention to detail and problem-solving** in technical tasks.
